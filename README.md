## Run Locally

Clone the project

```bash
  git clone https://github.com/HungDo007/video-editing
```

Go to the project directory

```bash
  cd client
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

In back - end,

```bash
  run in VS Studio
```
